O que é HTML?

A sigla HTML significa Hyper Text Markup Language, que significa Linguagem de Marcação de Hipertexto. É uma linguagem de marcação utilizada na construção de páginas na Web. O HTML é fácil de aprender, mas não é uma linguagem de programação. É um mecanismo de marcação para apresentar determinada informação em um navegador da web.As paginas podem conter textos,imagens animações, sons, etc

O que é CSS

A sigla css significa Cascading Style Sheets, que significa Folhas de Estilo em Cascata. É uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou em XML. O CSS é uma linguagem de folha de estilo composta por “camadas”, criado para estilizar as páginas HTML. O CSS separa o conteúdo da representação visual do site.

O que é Javascript

O javascript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é um das três principais tecnologias da World Wide Web. Muito usada para adicionar dinamismos nas páginas web, ou seja, atualizações em tempo real, mudanças em imagens, funções ativadas por iteração do usuario como onclick, .onmouseover, etc.
